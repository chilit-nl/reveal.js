Wanneer je nog niet bekend bent met de mogelijkheden van Spring Boot 3 is deze workshop ook geschikt voor jou.

De workshop
Onze workshop is gericht op het toepassen van Spring Boot 3 in de praktijk. Naast theoretische kennis zullen we ook hands-on opdrachten bieden waarbij je de nieuwe versie van het framework daadwerkelijk kunt toepassen. Zo leer je onder andere over Spring Problem details (de nieuwe standaard van het communiceren van fouten), Java interface clients, en hoe je een bestaand project kunt upgraden naar Spring Boot 3. Daarnaast behandelen we ook Observability met micrometer, hoe je je applicatie inzichtelijk maakt, en hoe Spring Native je kan helpen om je applicatie nog sneller te maken.
